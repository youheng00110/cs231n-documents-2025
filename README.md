# CS231N 2025年课程资料说明

本文件夹包含斯坦福大学CS231N课程2025年的全套学习资料，具体包括：
- 讲座PPT
- 课堂讨论内容
- 英文版作业
- 带注释的中文版作业


## 官方资源链接
- CS231N项目官网：[https://github.com/cs231n/cs231n.github.io](https://github.com/cs231n/cs231n.github.io)


## 数据集下载
### 直接下载
- CIFAR-10数据集：
  - 官网地址：[https://www.cs.toronto.edu/~kriz/cifar.html](https://www.cs.toronto.edu/~kriz/cifar.html)
  - 直接下载链接：[https://www.cs.toronto.edu/~kriz/cifar-10-python.tar.gz](https://www.cs.toronto.edu/~kriz/cifar-10-python.tar.gz)
- ImageNet子集：[http://cs231n.stanford.edu/imagenet_val_25.npz](http://cs231n.stanford.edu/imagenet_val_25.npz)


### 脚本下载方法
1. **安装Git工具**  
   下载并安装Git for Windows：[https://git-scm.com/download/win](https://git-scm.com/download/win)  
   安装时请确保勾选「Git Bash Here」选项，以便后续快速启动终端。

2. **执行下载脚本**  
   在脚本所在文件夹右键，选择「Git Bash Here」打开终端，依次执行以下命令：  
   ```bash
   chmod +x get_datasets.sh
   ./get_datasets.sh
   ```

   > ⚠️ 注意：Windows系统与Linux系统对应的脚本可能不同，请根据当前操作系统选择匹配的`get_datasets`脚本版本。
